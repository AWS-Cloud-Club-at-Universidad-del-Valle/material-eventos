# Lanzamiento Oficial: Tu Primer Paso en la Nube ☁️🚀

Este repositorio contiene el material compartido durante el **primer evento del AWS Cloud Club Universidad del Valle**, titulado:

**“Lanzamiento Oficial: Tu Primer Paso en la Nube”**

El objetivo del evento fue introducir a los estudiantes al ecosistema de **Amazon Web Services (AWS)** y motivarlos a dar sus **primeros pasos en la computación en la nube**.

---

# 🎯 Objetivos del evento

Durante este evento los participantes:

- Conocieron qué es **AWS** y por qué es importante aprender **computación en la nube**.
- Descubrieron algunas de las plataformas oficiales de aprendizaje de AWS.
- Recibieron material de referencia para crear sus cuentas y comenzar a aprender por su cuenta.

---

# 📚 Material compartido

Durante el evento se mencionaron y compartieron las siguientes guías para que los estudiantes puedan **crear sus cuentas posteriormente y comenzar su aprendizaje en AWS**.

## 1️⃣ Crear una cuenta en AWS Skill Builder

Guía para registrarse en la plataforma oficial de aprendizaje de AWS.

📄 Guía paso a paso:  
[crear-cuenta-skill-builder.md](crear-cuenta-skill-builder.md)

En esta plataforma podrás acceder a:

- Cursos de AWS
- Rutas de aprendizaje
- Preparación para certificaciones
- Laboratorios prácticos

---

## 2️⃣ Crear una cuenta en AWS (Free Tier)

Guía para crear una cuenta en **Amazon Web Services** utilizando el **Free Tier**, que permite utilizar varios servicios de AWS de forma gratuita dentro de ciertos límites.

📄 Guía paso a paso:  
[crear-cuenta-aws.md](crear-cuenta-aws.md)

Esto permite a los estudiantes:

- Crear recursos en la nube
- Desplegar aplicaciones
- Practicar con servicios reales de AWS

---

## 3️⃣ Crear una cuenta en AWS Educate

Guía para registrarse en **AWS Educate**, un programa diseñado para estudiantes que ofrece recursos educativos para aprender sobre la nube.

📄 Guía paso a paso:  
[crear-cuenta-aws-educate.md](crear-cuenta-aws-educate.md)
