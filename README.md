# 📚 Material de Eventos - AWS Cloud Club Universidad del Valle

Este repositorio contiene el **material oficial de los eventos, talleres y workshops** realizados por el AWS Cloud Club de la Universidad del Valle.

Aquí encontrarás recursos utilizados en nuestras actividades, como:

* 📊 Presentaciones (slides)
* 🧪 Guías de laboratorio
* 💻 Código utilizado en workshops
* 🖼️ Imágenes y material de apoyo
* 📄 Documentación adicional

El objetivo de este repositorio es **centralizar y compartir el conocimiento generado en los eventos del club**, para que cualquier estudiante pueda consultarlo, repasarlo o utilizarlo para aprender sobre tecnologías en la nube.

---

# 🗂️ Organización del repositorio

El contenido está organizado **por año y por mes** para facilitar la navegación.

Estructura general:

```
AÑO/
 └── MES/
      └── nombre-del-evento/
           ├── README.md
           ├── presentacion/
           ├── imagenes/
           ├── laboratorio/
           └── recursos/
```

Ejemplo:

```
2026/
 └── 03-marzo/
      └── lanzamiento-oficial-tu-primer-paso-en-la-nube/
           ├── README.md
           ├── presentacion/
           └── imagenes/
```

---

# 🔎 ¿Cómo encontrar el material de un evento?

Puedes buscar el contenido siguiendo estos pasos:

### 1️⃣ Buscar por año

Primero entra al directorio del año en que se realizó el evento.

Ejemplo:

```
2026
```

### 2️⃣ Buscar por mes

Dentro del año encontrarás carpetas con el mes del evento.

Ejemplo:

```
03-marzo
```

### 3️⃣ Buscar el evento

Dentro del mes encontrarás una carpeta con el nombre del evento.

Ejemplo:

```
lanzamiento-oficial-tu-primer-paso-en-la-nube
```

Dentro de esta carpeta encontrarás **todo el material relacionado con ese evento**.

---

# 🚀 Contribuciones

Si eres miembro del club y deseas agregar material de un evento:

1. Crea una carpeta siguiendo la estructura del repositorio.
2. Añade el material correspondiente (slides, guías, código, etc.).
3. Incluye un `README.md` explicando el contenido del evento.
